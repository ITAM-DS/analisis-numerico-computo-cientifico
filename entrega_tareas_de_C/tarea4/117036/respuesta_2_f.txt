¿Qué efecto tiene escribir extern para la declaración de variables dentro de main(), getline() y copia()?

Nota sobre extern
Para entender el efecto del comando extern es necesario notar la distinción entre declaración de variables y definición de las mismas. Definicion implica declaración, pues al definir una variable además de declarar el tipo de dato a alocar se indica, se reserva un espacio de memoria para almacenarla.
Extern para declarar funciones en C tiene dos principales usos: 1) Cuando se desea declarar variables pero no definirlas (tal vez por que otro programa ya las definió); 2) Para extender la visibilidad de la variable a todo el programa, cuando se usa permite el uso de la variable en cuestion en cualquier
parte del programa (suponiendo que se conoce el tipo de dato declarado y que ya haya sido definido en alguna otra parte)

En términos del ejercicio, la variable "linea_max" es una variable "extern" la cual conservará su declaración (tipo de variable) hasta que sea inicializada dentro de la funcion copia
