a. 
#include <file>
Se utiliza cuando los headers son archivos del sistema. Se busca el file en la lista estándar de directorios del sistema.
#include "file"
Se utiliza para headers del mismo programa. Se busca el file dentro del directorio que contiene al archivo actual.

b.
Una variable estática dentro de una función preserva su valor a través de las invocaciones.
Una variable estática global solo puede ser vista dentro del archivo en el que es declarada.
