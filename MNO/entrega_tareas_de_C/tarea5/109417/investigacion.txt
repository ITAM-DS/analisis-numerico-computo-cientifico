LAPACK:
LAPACK por sus siglas en ingles (Linear Algebra Package) es una libreria de software para algebra lineal, que provee
rutinas para resolver sistemas de ecuaciones lineales, minmos cuadrados, problemas de eigenvalores y descomposicion en
valores singulares. Tambien incluye rutinas que implementan factorizacion de matrices como LU, QR, Cholesky, etc. LAPACK
fue escrito inicialmente en FORTRAN 77 per ofue traducido a FORTRAN 90 en la version 3.2. Las rutinas estan listas para
manejar matrices reales y complejas en precision simple y doble.

BLAS:
Basic Linear Algebra Subprograms es una especificacion que pre escribe un conjunto de rutinas a bajo nivel para realizar
operaciones comunes de algebra lineal como: suma de vectores, multiplicacion de escalares, producto punto, combinaciones
lineales, y multiplicacion de matrices. Las rutinas tienen "bindings" para C y Fortran.

ATLAS:
Automatically Tuned Linear Algebra Software, es una libreria de software para algebra lineal que proporciona una 
implementacion madura de codigo abierto de las "api's" de BLAs y Fortran77. ATLAS es comunmente recomendado como una
forma de generar una libreria de BLAS optimizada. ATLAS corre en casi todos los sistemas operativos basados en 
UNIX y en Microsoft Windows usando Cygwin.

DGEMM:
DGEMM es una rutina de BLAS utilizada para realizar multiplicacion de matrices. Funciona con numeros reales de doble
precision.
